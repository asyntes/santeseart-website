# Santese Art — Sito Ufficiale

Elegante sito web minimal in bianco e nero per **Santese Art**, ebanisteria e falegnameria artistica di Rocco Santese a Oria (BR), Puglia.

![Santese Art](https://github.com/asyntes/santeseart-website/raw/main/public/og-image.jpg)

## ✨ Caratteristiche

- Design **minimal ed elegante** — solo bianco e nero
- Tipografia raffinata con Playfair Display + Geist
- Completamente responsive
- Navbar fissa con menu mobile
- Hero impactante
- Sezioni: Chi siamo, Servizi, Galleria, Contatti
- Form di contatto che apre il client email
- Immagini Unsplash ottimizzate
- Costruito con **Next.js 16** + Tailwind CSS 4

## 🚀 Deploy

Il sito è collegato a **Vercel** su **santeseart.vercel.app**

```
git clone https://github.com/asyntes/santeseart-website.git
cd santeseart-website
npm install
npm run dev
```
