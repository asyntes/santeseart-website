# Santese Art — Sito Ufficiale

Elegante sito web **mobile-first**, minimal e moderno per **Santese Art**, ebanisteria e falegnameria artistica di Rocco Santese a Oria (BR), Puglia.

![Santese Art](https://github.com/asyntes/santeseart-website/raw/main/public/og-image.jpg)

## ✨ Caratteristiche

- Design **mobile-first, minimal ed elegante** — solo bianco e nero, ottimizzato per tutti i dispositivi
- Tipografia raffinata e moderna: **Instrument Serif** (elegante, minimal) + **Geist** (pulito e contemporaneo)
- Completamente responsive con navbar fissa e menu hamburger mobile fluido
- Hero impactante con tipografia forte
- Sezioni: Chi siamo, Servizi, Galleria, Contatti
- Form di contatto che apre il client email
- Immagini Unsplash ottimizzate e performanti
- Costruito con **Next.js 16** + **Tailwind CSS 4** (mobile-first utility classes)

## 🚀 Deploy

Il sito è collegato a **Vercel** su **santeseart.vercel.app** (deploy in corso / verifica build)

```
git clone https://github.com/asyntes/santeseart-website.git
cd santeseart-website
npm install
npm run dev
```
